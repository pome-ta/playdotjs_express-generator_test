# playdotjs_express-generator_test

play.js で`express-generator` を試してみるリポジトリ


## 手順

Working Copy も使ってる

1. Github 上でリポジトリつくる
1. Working Copy でclone
1. play.js でWorking Copy のディレクトリをコピー




``` package.json
{
  "name": "playdotjs_express-generator_test",
  "version": "1.0.0",
  "scripts": {
    "init": "express"
  },
  "dependencies": {
    "express-generator": "4.16.1"
  }
}
```
